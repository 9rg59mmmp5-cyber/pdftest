# pdftest

KPSS Onlisans hazirlik uygulamasi. PDF'ten soru/not cozumu, bilimsel tekrar, ezber kartlari, YouTube ve Telegram entegrasyonu.

## Servisler

- pdftest-api — Node.js Express, port 4001
- pdftest-ytgen — Python FastAPI, port 4002 (YouTube -> ezber karti)
- pdftest-tgbot — Python, port 4003 (Telegram -> soru kaydi)

## Dizin

- frontend/ — React TypeScript Vite (App.tsx)
- backend/ — Node.js
- ytgen/ — Python FastAPI
- tgbot/ — Python Telegram bot
- install/ — Kurulum scriptleri
